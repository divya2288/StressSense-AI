# 🧠 StressSense AI - Mental Wellness Platform

> AI-powered stress detection and mental wellness management system

## 🌟 Features
- 📝 AI Text Analysis - Detects stress from written input
- 📋 Burnout Assessment - ML-based 6-question evaluation
- 🔥 Body Stress Heatmap - Interactive physical stress mapping
- 🤖 AI Chatbot Therapist - 24/7 intelligent support
- 🎵 Meditation Player - Guided audio sessions
- 🧘 Breathing Exercise - 4-7-8 relaxation technique
- 📊 Weekly Analytics - Progress tracking and insights
- 💡 Personalized Wellness Recommendations

## 🛠️ Tech Stack
- **Frontend:** React.js | JavaScript | HTML5 | CSS3
- **Backend:** Flask | Python | REST API
- **ML/AI:** NLP | Sentiment Analysis | Scikit-learn
- **Tools:** Chart.js | SVG Graphics | Web Speech API | Git

## 🚀 How to Run

### Frontend
```bash
cd stress-frontend
npm install
npm start
```

### Backend
```bash
cd stress-backend
pip install -r requirements.txt
python app.py
```

### ML Service
```bash
cd stress-ml
pip install -r requirements.txt
python app.py
```
